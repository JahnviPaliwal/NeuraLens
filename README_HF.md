# NeuraLens — Deep Learning Layer Explorer

Interactive, full-stack explorer for 60+ deep learning layers with real backend computation, live Canvas visualizations, hardware suitability analysis (GPU/FPGA/ASIC), AI-generated explanations, a layer comparator, and an architecture builder.

Built with Express (Node.js) + vanilla JavaScript frontend. White/red/blue design.

See the main `README.md` in this repo for full architecture details.
